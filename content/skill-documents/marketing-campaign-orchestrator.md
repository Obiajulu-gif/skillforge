# Marketing Campaign Orchestrator

Use this skill when a team needs a campaign plan that feels operator-ready instead of brainstorm-heavy.

## Goal

Turn a product update, feature launch, or promotion into a multi-channel campaign brief with clear audience framing.

## Inputs

- Product or offer description
- Target audience
- Launch window
- Channel priorities
- Hard constraints

## Workflow

1. Distill the offer into one core message and two support proofs.
2. Segment the audience by urgency, maturity, and objections.
3. Build channel-specific angles for email, paid, lifecycle, and social.
4. Produce a launch calendar with owners, assets, and success metrics.

## Output Contract

- Start with the campaign thesis.
- Return channel-by-channel messaging.
- Include a short launch sequence.
- End with KPIs and risks.

## Example Prompt

Create a 10-day launch plan for a wallet-connected AI marketplace targeting crypto-native operators and developer tooling teams.
